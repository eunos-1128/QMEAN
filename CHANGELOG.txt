Changes in Release 4.X.X
--------------------------------------------------------------------------------

 * GMQE (Global Model Quality Estimate) to predict expected quality of a protein
   model given a certain template structure.
 * Several minor bug fixes, improvements

Changes in Release 4.0.0
--------------------------------------------------------------------------------

 * First release supporting Python 3. Python 2 support has been dropped.
 * Several minor bug fixes, improvements

Changes in Release 3.2.0
--------------------------------------------------------------------------------

 * This is expected to be the last release supporting Python 2.
 * Added automated testing of all example scripts in the documentation.
 * Several minor bug fixes, improvements

--------------------------------------------------------------------------------

 * Version string available as qmean.qmean_version or qmean.qmean_version_x

 * Several minor bug fixes, improvements
